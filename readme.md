# 🛡️ 面向零信任网络的 APT 加密恶意流量智能感知与溯源系统

## 1. 项目简介
本项目为第二十一届中国研究生电子设计竞赛参赛作品。
系统打通了从底层 PCAP 报文解析、多模态特征融合提取，到高维 GATv2 图神经网络推理的完整闭环。无需离线导入，支持流式交互，并提供具备可解释性（XAI）的态势感知前端面板。

## 2. 核心文件说明
- `app.py`：系统主入口，包含 Streamlit 前端大屏渲染逻辑。
- `inference_engine.py`：后端核心推理引擎，封装了 MLP、CNN 与 GATv2 的前向推理与 KNN 动态构图逻辑。
- `pcap_feature_extractor.py`：流量解析探针，利用 Scapy 提取 50 维统计特征与真实香农熵。
- `*.pt` / `*.pkl` / `*.npz`：系统所需的预训练模型权重与常驻内存的知识图谱锚点库。
- `sample_pcaps/` 目录：存放了用于系统测试的真实流量切片（包含正常流量与 APT 攻击流量）。
